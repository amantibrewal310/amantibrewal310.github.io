# amantibrewal310.github.io

Personal portfolio — live at [amantibrewal310.github.io](https://amantibrewal310.github.io).

Plain HTML/CSS/JS, no build step. Design direction: an "annotated document" —
paper, ink, highlighter marks, and a hero that renders the bio as a source
document beside its extracted, schema-valid JSON profile. Set in IBM Plex.

## Run locally

Open `index.html` in a browser. That's it.
